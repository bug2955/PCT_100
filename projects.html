实训项目

项目一：温度采集与风扇控制

难度：★★☆☆☆ 入门级
学习 DS18B20 单总线通信协议，实现温度数据采集； 设定滞回阈值，通过继电器驱动风扇实现自动温控。 掌握传感器驱动、执行器控制、阈值逻辑编程。

项目二：光照采集与灯控

难度：★★☆☆☆ 入门级
学习 BH1750 I2C 通信协议，采集环境光照强度； 根据光照阈值自动控制继电器开关灯。 掌握 I2C 总线通信和双阈值控制算法。

项目三：OLED 显示与人机交互

难度：★★★☆☆ 中级
驱动 SSD1306 OLED 显示温度、光照、继电器状态； 实现按键扫描与多页面菜单切换。 掌握人机交互界面（HMI）设计和状态机编程。

项目四：WiFi 与 MQTT 物联网通信

难度：★★★★☆ 中高级
配置 ESP32 WiFi 连接，通过 MQTT 协议上报传感数据； 使用 JSON 格式封装数据包，实现远程指令下发控制。 掌握物联网三层架构和 MQTT QoS 机制。

项目五：FreeRTOS 多线程编程

难度：★★★★☆ 中高级
创建多个 FreeRTOS 任务：网络通信、LED 渲染、主控逻辑； 使用 xQueue 实现任务间安全通信。 掌握实时操作系统（RTOS）任务调度、优先级、队列等核心概念。

项目六：多回路综合控制系统

难度：★★★★★ 高级
整合温度、光照两个控制回路，实现协同工作； 本地 OLED 显示 + MQTT 远程监控双重界面； RGB LED 综合状态指示。 全面掌握嵌入式系统设计与过程控制原理。

推荐学习路径

1

基础入门（第 1-2 周）

完成项目一和项目二，学习传感器驱动和继电器控制，掌握 Arduino 编程基础。

2

人机交互（第 3 周）

完成项目三，学习 OLED 显示驱动和按键交互，掌握用户界面设计方法。

3

物联网进阶（第 4-5 周）

完成项目四，学习 WiFi 和 MQTT 通信，搭建云平台数据看板。

4

系统深化（第 6-7 周）

完成项目五和项目六，深入学习 FreeRTOS 和多回路协同控制设计。
